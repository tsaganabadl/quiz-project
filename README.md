Литературная викторина

Викторина на HTML, CSS и JavaScript. 10 вопросов по русской и зарубежной литературе.

Структура проекта:

quiz-project/
|-- index.html
|-- css/
│   |-- style.css
|-- js/
|   |-- main.js
|-- README.md

Как запустить: откройте файл index.html в любом браузере.

Что реализовано:
- 10 вопросов с вариантами ответов
- Прогресс-бар
- Подсветка правильного ответа зелёным, неправильного красным
- Подсчёт баллов
- Кнопка «Результаты» с экраном результатов
- Кнопка «Начать заново»
